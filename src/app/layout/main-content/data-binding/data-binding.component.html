1. What is data binding?

Data binding keeps your template in sync with the component class. Angular supports:

  • Interpolation — {{ syntax.interpolation }}
  • Property binding — {{ syntax.property }} (DOM properties, directives, [attr.name], [class], [style])
  • Event binding — {{ syntax.event }}
  • Two-way binding — {{ syntax.twoWay }} (forms; needs FormsModule)
Styling below uses Bootstrap 5 utility classes only (e.g. btn, alert, bg-dark).

3. Property binding

Square brackets bind a template expression to a property on the host element.

[value] — one-way into the input
[class] — Bootstrap alert classes
Success state via bound class string.
[type] — toggle native input type
[class] — theme panel (Bootstrap utilities)
