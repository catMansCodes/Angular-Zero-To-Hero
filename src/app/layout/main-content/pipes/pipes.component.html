1. What are pipes?

Pipes transform data in the template for display without mutating the source value. Use the pipe character | followed by the pipe name and optional parameters.

Typical syntax: {{ syntaxPipe }} — the table below shows live output for common pipes.

Common built-in pipes
  • date, currency, decimal, percent
  • uppercase, lowercase, titlecase, json
  • async — unwraps Observables and Promises
  • Custom pipes — implement PipeTransform for your own rules

3. Pure vs impure pipes

By default pipes are pure: Angular runs them when a primitive input or object reference changes. Impure pipes run every change detection cycle—use sparingly.

Guidance
  • Prefer pure pipes and immutable data for predictable performance.
  • Use impure only when you must react to deep mutations (and measure cost).
